# Madhav-Store-Dataset
Madhav Store: A Dynamic Sales Dataset for Growth

About Dataset

**Orders CSV Dataset**
Description:
The "Orders CSV" dataset provides valuable insights into the ordering process of a retail company. It includes information related to customer orders, such as order ID, order date, customer name, state, and city. This dataset is a key resource for analyzing customer behavior, geographical distribution, and order history.

Columns:
Order ID: A unique identifier for each order.
Order Date: The date when the order was placed.
Customer Name: The name of the customer placing the order.
State: The state in which the customer resides.
City: The city where the customer is located.

**Details CSV Dataset**
Description:
The "Details CSV" dataset complements the "Orders CSV" dataset by providing detailed information about the orders. It includes data on order ID, amount, profit, quantity, category, sub-category, and payment mode. This dataset is crucial for analyzing sales performance, profitability, product categories, and payment methods.

Columns:
Order ID: A unique identifier matching the order with the "Orders CSV" dataset.
Amount: The total amount of the order.
Profit: The profit generated from the order.
Quantity: The quantity of products ordered.
Category: The broad category to which the product belongs.
Sub-Category: A more specific sub-category of the product.
Payment Mode: The mode of payment used for the order.
These two datasets, "Orders CSV" and "Details CSV," together offer a comprehensive view of the company's order data, enabling data-driven analysis and insights into customer behavior, geographical patterns, sales performance, and product categorization. Researchers and analysts can leverage these datasets to extract valuable information and support decision-making processes.
